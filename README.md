# SportStream - Live Sports Streaming Website

A fully responsive sports streaming website with admin panel functionality.

## Features

- **Live Streaming**: Watch embedded YouTube/Facebook streams
- **Responsive Design**: Works on all devices
- **Admin Panel**: Add, edit, and delete streams
- **Category Filtering**: Filter streams by sport type
- **Real-time Viewer Counts**: Dynamic viewer statistics
- **Upcoming Events**: Schedule of future matches
